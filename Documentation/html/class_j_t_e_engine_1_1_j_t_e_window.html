Detailed Description

This allows for the creation and manipulation of a window.

This is the foundation of a program. Everything that will be seen and rendered will be shown on this window.

Returns the FPS and the time between each frame.

This allows for debugging and helps with the optimization proccess or just fun information to look at.

Parameters
*fps- Where the FPS is being returned.
*milliseconds- Where the milliseconds are being returned.
Note
If outputted into the character stream, this will reduce performace as with any other debugging element.
The frames and ms counts updates every 1/30 of a second.

This used in the mainloop to tell when the loop should stop.

Returns

false - If window is not closing.
Note

void JTEEngine::JTEWindow::terminate ( )

Destroys the window resources and terminates GLFW.

This frees up resources and is good pratice to use this function. It ensures that resources are properly freed on termination.

Note
Make sure not to put this in the mainloop.
For your reference, all terminate or destroying functions should be called outside of the mainloop unless instructed.
