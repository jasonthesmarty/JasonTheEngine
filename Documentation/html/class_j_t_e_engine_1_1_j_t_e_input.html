Detailed Description

This allows for the use of input for different events.

If-statements are used to check if the specified button is pressed or not.

Note
This can only be used once as input checking are universal, even with multiple windows.

◆ keyDown()

JTEboolean JTEEngine::JTEInput::keyDown ( JTEuint  key)

This returns a value if the mouse button specified was pressed.

This checks if it was pressed and outputs a boolean value. It outputs a repeated stream of values. So if the button was only pressed for one second, it could output more than one press. This happens because it is in the mainloop.

Parameters
key- The mouse button being checked for input.
Returns
true - If the mouse button was pressed.
false - If the mouse button wasn't pressed.
Note
Toggle switches would have to be coded manually.
